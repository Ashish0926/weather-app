
# Weather App

A web app that uses Rapid Api to get weather and related information for the location entered.




## Run Locally

Clone the project

```bash
  git clone https://github.com/Ashish0926/weather-app.git
```

Go to the project directory

```bash
  cd weather-app
```

Use Live Server on vscode.
